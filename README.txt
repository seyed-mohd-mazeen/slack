How to start a project
1. Folder structure - index & style
2. create all the required files :- index.html, index.css, base.css, layout.css, form-field.css.
3. Code the required styles in their respected files.
4. Finding the colors, font, font-size, padding, font-weight and applying them their respected files. mainly in base.css
5. Finding the types of buttons and their colors and extra button size. (Primary and secondary button)
6. Designing the layout in HTML and adding comments to them.
7. Saving images from the webpage into image folder.
